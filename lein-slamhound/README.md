# lein-slamhound

Leiningen plugin to use Slamhound, a tool for reconstructing ns forms
from scratch.

## Usage

Add `lein-slamhound "1.3.0-SNAPSHOT"` to your `:plugins`, then run
`lein slamhound` in a project.
